# emi-calc

### *Simple EMI Calculator*

#### - Prerequisite
  -- NodeJs (v12.4.0+)


### - Run
- Install dependencies
  - `$ npm i`
- To run *dev-server*
  - `$ npm start`
- To generate build
  - `$ npm run build`
- To run *prod-server* (This generates prod build and starts `express` server)
  - `$ npm run prod`
